Паттерны рынка

A. Топ-5 практик

  • Низкий барьер входа — не скидка, а опыт (€50 intro, Founder Appointment)
  • Membership capped / numbered — scarcity без агрессивного маркетинга
  • Протоколы как named результаты, не список услуг
  • Диагностика как gateway, не дополнение к процедурам
  • Прозрачность цен — конкурентное преимущество (6 из 11 публикуют)

B. Возможности для REjuve

  • B2B — упущен почти всеми (только 3 из 11 имеют публичный B2B)
  • "Executive Pass" для нерегулярных клиентов (Zug expats)
  • Creative Room / meeting + wellness crossover
  • Membership transfer снижает страх lock-in

C. Риски для REjuve

  • Псевдонаучная репутация — критика Karolinska (V Social Club)
  • Операционная нагрузка float + cryo (Renessence убрала)
  • "Germany's First" легко оспорить — не делать без проверки
  • Слабый UX сайта при высоком price-point
  • Слишком много модальностей при запуске (Sauna & Plunge)

Что НЕ делать (anti-patterns)
1. "Social wellness space" языкMass-market язык (£130/mo ARPU). У REjuve price-point €300+/mo — слово "social" размывает positioning.
2. "Opening soon" без конкретной датыAEG ON: сайт работает с 2026, открытие H1 2029. Dilutes credibility у HNWI аудитории.
3. "First in X" без проверкиMitoSphere заявляет "Germany's First", при этом Recover Society уже во Франкфурте. Безопаснее: "единственный [differentiator] в Центральной Швейцарии".
4. Слабый UX сайта при высоком price-pointV Social Club — UX 2020 уровня снижает доверие у нового трафика. Сайт = часть premium positioning.
5. Перегрузить старт операционно сложными модальностямиRenessence убрала cryo и float. Sauna & Plunge: 6 plunge pools "операционно сложно". На старте: 4–5 core modalities.
6. Tier без понятного single outcomeOne Thirty Labs Optimised Health tier перегружен опциями без clear outcome. Каждый tier = 1 результат.
7. Все цены "по запросу"ETERNA: отсутствие прозрачности фрустрирует холодный трафик. Даже "от 300 CHF/mo" лучше пустого места.
