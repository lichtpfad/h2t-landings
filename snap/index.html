context

The Problem

AI agents see browsers (Playwright) and talk to APIs. But they're blind to desktop applications — Photoshop, Houdini, TouchDesigner, Excel, any native software.

h2t-snap gives them sight and touch. Download one .exe and your agent can see any window, decide what to do, act, and verify the result.

Capture → See → Act → Verify

quick start

30 Seconds to First Screenshot


# See what's running
$ h2t-snap list
[{"index": 0, "hwnd": 12345, "title": "TouchDesigner - project.toe"}]

# Screenshot a window
$ h2t-snap capture --title "TouchDesigner" --json
{"path": "C:\\...\\snap_20260328.png", "width": 1920, "height": 1080}

# Click somewhere
$ h2t-snap click --title "TouchDesigner" --x 450 --y 320

# Verify the result
$ h2t-snap capture --title "TouchDesigner"
