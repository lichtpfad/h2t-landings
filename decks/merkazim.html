// трек 0 · обязательный pre-session

Intro + Контекст

2 встречи × 3 часа

Контекст-менеджмент — большая тема, за 30 минут не уложить. Делим intro на две сессии: сначала снимаем техническую тревогу, потом учим работать с ограниченным ресурсом внимания модели.

SESSION 1

Установка

  • Глоссарий AI: prompt, agent, skill, hook, MCP, RAG
  • Установка: Claude, Claude Code, GitHub, git, venv
  • Первый запуск: hello-world + commit
  • Практика: базовые промпты, git-flow
SESSION 2

Контекст

  • Контекст как ограниченный ресурс
  • Длинные документы: chunking, summaries, RAG
  • Skills, hooks, memory в Claude Code
  • Субагенты и session hygiene
  • Практика: свой шаблон context hygiene
